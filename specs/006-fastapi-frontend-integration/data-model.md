# Data Model: Chat Interaction

**Version**: 1.0
**Status**: DRAFT
**Author**: Gemini Agent
**Created**: 2025-12-31
**Last Updated**: 2025-12-31

## 1. Chat Request

Represents a query sent from the frontend to the backend.

-   **Entity**: `ChatRequest`
-   **Fields**:
    -   `query` (string, required): The user's question.
    -   `selected_text` (string, optional): The text selected by the user in the book.
-   **Validation**:
    -   `query` must not be empty.

## 2. Chat Response

Represents the response sent from the backend to the frontend.

-   **Entity**: `ChatResponse`
-   **Fields**:
    -   `answer` (string, required): The answer generated by the RAG agent.
    -   `source` (string, optional): The source from the book used to generate the answer.
-   **Validation**:
    -   `answer` must not be empty.
